# CornShell-AI
Smart corn shelling analytics system using OpenCV, machine learning, and sensor-based data acquisition for predicting shelling performance and losses

## Features

- Corn loss estimation using OpenCV
- IoT sensor integration
- Random Forest Regressor prediction
- Shelling performance analytics

## Hardware

- Arduino UNO
- Load Cell + HX711
- Hall Effect Sensor
- IR Sensor
- GMA
- Camera

## Dataset Features

Input:
- Corn Weight
- RPM
- Moisture Content

Output:
- Processing Time
- Yield Weight
- Losses

## Technologies

Python
OpenCV
Scikit-learn
Arduino
ESP32
